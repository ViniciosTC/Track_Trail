# 💰 Sistema de Controle Financeiro

Aplicação em Java para gerenciamento de despesas pessoais, com controle de orçamento e análise da saúde financeira.

---

## 📌 Funcionalidades

- Cadastro de despesas com descrição e valor  
- Categorização automática de despesas  
- Definição de renda mensal  
- Sugestão automática de limite de gastos (70% da renda)  
- Controle de orçamento em tempo real  
- Relatórios financeiros  
- Persistência de dados com SQLite  

---

## 🧠 Conceitos aplicados

Este projeto foi desenvolvido aplicando conceitos de orientação a objetos e padrões de projeto:

- Facade → centralização da lógica do sistema  
- Strategy → categorização flexível de despesas  
- Observer → monitoramento do orçamento  
- Singleton → conexão com banco de dados  

---

## 🛠️ Tecnologias utilizadas

- Java  
- SQLite  
- JDBC  
- Git e GitHub
